# dotfyle
To update, try:
```{zsh}
zsh run.sh -u 1
```

To install to a new machine, try:
```{zsh}
zsh run.sh -i 1
```
Configuration to be sync:
- .vimrc
- .zshrc
- .tmux.conf
- .config/clangd
- .config/sioyek
- .wezterm.lua

Things to be installed:
- neovim
- clang-format
- cppcheck
- cpplint
- ripgrep
- black
- oh-my-zsh and configs

Enjoy!
